Тестирование создания медведя (createBear):

   1 Проверка успешного создания медведя с корректными данными.
   2 Проверка создания нескольких медведей разных типов.

Тестирование получения и удаления медведя по его ID (getBearById/deleteBearById):

   1 Проверка получения медведя по существующему ID.
   2 Проверка удаления медведя по существующему ID.

Тестирование обновления медведя по его ID (updateBearById):

   1 Проверка успешного обновления медведя с корректными данными.
